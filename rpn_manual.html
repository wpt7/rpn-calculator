#RPN 计算器使用说明书

##1. 安装和编译指南

###安装系统

Linux/Unix 系统

  • GCC 编译器
  • Nim 语言环境（用于生成文档）

###编译

编译 C++ 程序 g++ rpn_calc.cpp -o rpn_calc -lm

###运行程序 ./rpn_calc

##2.基本使用示例

编译 C++ 程序 g++ rpn_calc.cpp -o rpn_calc -lm

运行程序 ./rpn_calc

输入表达式:操作数1 操作数2 操作符(例, '5 5 +'), 或 'q' 退出.

##3.所有支持的操作说明

这是一个基于 C++ 实现的逆波兰表示法（Reverse Polish Notation）计算器，支持基础四则运算、高级数学函数(平方根、幂运算及斐波那契数列等)和错误处理(处理除零错误 处理输入格式错误 处理栈空错误)。 四则运算支持

加法 (+) 减法 (-) 乘法 (*) 除法 (/)

栈操作功能

支持数字输入（整数和浮点数） 支持基本的栈操作：清空、显示当前栈 支持连续运行操作: 将输入的rpn语法字符串的最终计算结果输出

错误处理

处理除零错误 处理输入格式错误 处理栈空错误

##4.错误代码和异常说明

每次测试栈空错误前要先清空栈

clear # 清空栈 5 + # 测试栈空错误 错误：栈空，需要两个操作数

clear # 清空栈

  •        # 测试完全空栈

错误：栈空，需要两个操作数

##5.示例输入输出 输入表达式 (例, '5 5 +'), 或 'q' 退出.

1 2 + 3 + 7 7 * + 结果: 55
